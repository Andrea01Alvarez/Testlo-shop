# Teslo API

1. clonar proyecto
2. ```yarn install```
3. clona el archivo ```.env.template``` y renombrar a ```.ev```
4. cambiar las variables de entorno 
5. Levantar la base de datos
```
docker-compose up -d
```
6. Ejecutar SEED
```
http://loscalhost:3000/api/seed
```

6. Levantar: ```yarn start:dev```
